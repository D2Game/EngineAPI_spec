Size
====

Public methods
----
* new(width, height)

  Size 오브젝트를 생성한다.
  width : 가로 크기 (Fixnum)
  height : 세로 크기 (Fixnum)
  
* +(size)

  Size 오브젝트끼리 더하기를 수행한다.
  size : 더할 Size 오브젝트
  
* -(size)

  Size 오브젝트끼리 빼기를 수행한다.
  size : 뺄 Size 오브젝트
  
Properties
----
* width[RW]
  
  가로 크기 (Fixnum)
  
* height[RW]

  세로 크기 (Fixnum)
