Color
====

Constants
----
* White : rgb(255,255,255)
* Black : rgb(0,0,0)

Public methods
----
* __new(r,g,b,a=255)__
  
  Color 오브젝트를 생성한다.<br>
  r : Red 값 (Finxnum)<br>
  g : Green 값 (Fixnum)<br>
  b : Blue 값 (Fixnum)<br>
  a : Alpha 값 (Fixnum)
  
* __+(color)__

  Color 오브젝트끼리 더하기를 수행한다.<br>
  color : 더할 Color 오브젝트
  
* __-(color)__

  Color 오브젝트끼리 빼기를 수행한다.<br>
  color : 뺄 Color 오브젝트
  

Properties
----

  Red 값 (Fixnum)
  
* __g__[RW]

  Green 값 (Fixnum)
  
* __b__[RW]

  Blue 값 (Fixnum)
  
* __a__[RW]

  Alpha 값 (Fixnum)
